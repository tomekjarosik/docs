LXC containers
====

List of available containers:
https://us.images.linuxcontainers.org/

How to start a new container:
```lxc launch ubuntu:18.04/amd64 my-container```

Then:
`lxc list` and 
```
lxc exec my-container bash
```
### Storage
`lxc storage list`
